# vim-hs-sort-imports

Sort Haskell import statements in Neovim the way I like.

## Prerequisites

* Neovim.  Tested on 0.6.x.

## Installation

I use [vim-plug](https://github.com/junegunn/vim-plug).

```vimscript
call plug#begin()
    Plug 'sjshuck/vim-hs-sort-imports'
call plug#end()
```

`:PlugUpdate`
